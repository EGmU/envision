## Required ##
Class `sv-spinner` 

```javascript
   // Instantiate
   $('mySpinner').spinner();
```

```javascript
   // Destroy
   $('mySpinner').spinner('destroy');
```

```javascript
   // Hide
   $('mySpinner').spinner('hide');
```

```javascript
   // Show
   $('mySpinner').spinner('show');
```
